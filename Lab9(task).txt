1. Создать структуру STUDENT{ФИО, №Группы, массив из 5 отметок}
2. Создать программу, которая:
a) Хранит введенные с клавиатуры данные для каждой структуры STUDENT в массиве;
б) Сортирует данные по возрастанию по №Группы;
в) Выводит ФИО и №Группы только для студентов чей (средний балл >=4);
г) Выводит в виде таблицы, столбцы: «Номер группы», «Количество студентов», «Количество двоечников».Строки таблицы должны быть упорядочены по убыванию значений в столбце «Количество двоечников».
